屋久島は年間降水量4,000〜8,000mm（山間部）。この降水量を資源として設計する。

前提：屋久島の水環境データ

  • 超軟水（硬度10ppm程度）・弱酸性
  • 関東の硬度40〜60ppmと比べて浄化コストが低い
  • 活性炭ろ過＋簡易UV殺菌で生活用水レベルに到達しやすい
  • 屋久島縄文水など市販ブランドが成立するレベルの水質

エネルギー連動（重要）

  • 屋久島電工が水力発電で島内電力の99.6%を自給している先行事例あり
  • EV急速充電を無料提供（発電過剰時の活用）
  • 「脱炭素に一番近い島」として全国から注目

1. 集水技術

屋根集水が最も効率的な出発点。
設計式: 集水量 = 降雨量 × 屋根面積（水平投影） × 流出係数（屋根: 0.9〜1.0）

構成要素選択肢
初流水除去ファーストフラッシュダイバーター（降雨初期の汚濁物を自動排除）
貯留タンクピロティ下地下埋設型 / 大型地上型（1,000L〜50m³）
補助水源沢水・湧水との2系統設計

事例

  • 「雨とともに暮らす家」（GQハウス）: 2tの地下タンク、4人家族の1日750Lを確保。砂・活性炭多段ろ過＋UV殺菌で飲料水・生活用水を同一系統で供給。
  • 沖縄「りっか！雨水利用」: 渇水地域の教訓から設計ガイドとExcelシミュレーターを無料提供。個人〜施設向けに展開。

屋久島への示唆

  • 年間降水量が関東の3〜5倍 → 集水効率が圧倒的に有利
  • 壁面流下雨水も屋根面積に加算する設計を検討
  • 防火用水との兼用（消防水利への算入可能性）
  • メンテナンスは民間委託可能な設計にする（島内業者 vs 本土業者のコスト比較が必要）

2. 浄化・水質管理

多段ろ過の基本構成

小石層 → 砂利層 → 活性炭層 → 砂層 → UV殺菌
用途必要浄化レベル検査義務
飲料水51項目 (水道法)厳格
生活用水（洗面・風呂）簡易浄化で可自治体条例
農業用水基本的に規制なしなし
トイレ・散水（雑用水）水道法適用外4〜11項目程度

屋久島の優位性

  • 超軟水・弱酸性 → 石灰質ろ過層が不要、浄化時間を短縮できる
  • 浄化フィルターの使用済み砂・活性炭は庭園土壌改質材として再利用可

実用的な設計提案

  • 2系統分離: 飲料水系（多段ろ過＋UV） + 雑用水系（簡易ろ過のみ）
  • UV殺菌ランプ: 6〜12ヶ月交換。DIY交換できる製品選定が重要
  • 水質検査: 鹿児島県保健所または民間検査機関で月1〜3回（3,000〜8,000円/回）
  • 住民向け簡易検査キット（pH・濁度・残留塩素）を常備

3. 流域治水・グリーンインフラ

単に「雨水を集める」のではなく、敷地全体で雨水の流れをデザインする。

屋根集水 → 雨庭 → スウェイル（等高線沿い浸透溝）→ 地下水涵養 → 湧水復帰
               ↓
          多段調整池（土砂沈降）→ 農業用水・雑用水

雨庭（Rain Garden）

  • 降雨後24〜48時間で完全浸透
  • 屋根面積の1/5〜1/10の面積が目安
  • 実験値で屋根雨量の約63%を削減
  • 建物から1.5〜3m離して設置（安全距離）

スウェイル（浸透溝）

  • 等高線に沿って掘削。底部勾配0.5〜2%、幅1〜2m、深さ30〜60cm
  • パーマカルチャーのコア技術
  • 屋久島の急峻地形：等高線設計が地形と完全にマッチ

山地・濁水対策（屋久島特有の課題）

  • 屋久島花崗岩は保水しない急峻地形 → 降雨の急速流出が起きやすい
  • 土砂沈降池を多段設置（高標高→中標高→敷地内）
  • 台風・豪雨時: 時間雨量100mm超を設計ベースラインに設定（1/50〜1/100年確率）

4. 業法・許認可・補助金

区分条件手続き
雑用水トイレ・散水・洗車用途原則不要（自治体条例確認）
簡易専用水道貯水タンク10m³超屋久島町への届出 + 年1回水質検査
専用水道100人以上に給水都道府県許可 + 厳格な検査義務

→ villageがタンク10m³超を保有する場合、簡易専用水道として届出が必要。
→ 飲料水を想定しないなら「雑用水」範囲に留めれば許認可を大幅に簡略化できる。

屋久島固有の規制

  • 世界遺産エリア（10,747ha、全島面積の2割）: 自然公園法・自然環境保全法・文化財保護法が重複適用
  • ただし、世界遺産バッファゾーンは設定なし → 登録地域外は相対的に緩い
  • 国立公園特別地域に該当する場合は環境省協議が必要

5. 小水力発電との連携

  • 屋久島電工: 昭和27年から安房川水系で水力発電。現在3カ所の発電所
  • 水力発電で島内電力の99.6%を自給
  • 落差（ヘッド）×流量で発電量が決まる
  • 導入コスト: 174〜293万円/kW（100kWで5,000万〜1億円）

設計アイデア

  • 雨水タンク → 小水力 → 農業用ポンプ駆動（直流給電で効率化）
  • 水力（雨季）× 太陽光（乾季）のハイブリッド自給モデル
  • 屋久島電工との連携協定で技術・許認可ノウハウを活用

6. 農業・食料生産との連携

雨水 → かんがい → 農業生産

      ↓

  • 国内で太陽光＋雨水を活用した唐辛子×チョウザメ飼育の自立型施設が実稼働（令和3年度事例）

屋久島villageに刺さるポイント（総まとめ）

圧倒的な優位性

  1. 年間降水量が関東の3〜5倍 → 集水量で他地域が羨む水資源
  2. 超軟水 → 浄化コストが低い、発酵食品に最適
  3. 小水力発電の先行モデルが島内にある（屋久島電工）→ 技術・許認可のロールモデル
  4. 屋久島花崗岩の急峻地形 → スウェイルとカスケード水力に適した地形

設計の優先順位

  1. 立地選定で降水量を決める（海岸部2,500mm vs 山岳部6,000mm）
  2. 飲料水化するか雑用水に留めるかで許認可が変わる → 段階的に高度化する設計が現実的
  3. 雨水×小水力の統合が屋久島の独自性（他地域では再現不可能）
  4. オストロムの8原則を設計図に組み込む → コミュニティ持続性の根拠になる
