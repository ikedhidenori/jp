CHAPTER 01

「匂う」を化学で
定義する

EP.01では「匂い＝空気にとびだす中性分子」とザックリ言った。今回は 何が「空気にとびだす」を決めるか を式で押さえよう。

嗅覚を司るのは 嗅覚受容体 (Olfactory Receptor; OR)。GPCR ファミリーの 7 回膜貫通タンパク質で、ヒトは約 400 種を持つ。OR は気相分子をリガンドとして認識するから、空気にまで漂ってこない分子はそもそも検知の土俵に上がれない。

「とびだせる」分子の条件、整理するとこの 4 つ:

①

分子量 30〜300 Da

大きすぎると気体になれない。タンパク質や糖鎖は無臭。

②

電気的に中性

イオンは水和殻が厚く気化不能。電荷を持たないこと。

③

logP ≳ 1.5

受容体ポケットの脂質環境に入り込める疎水性。

④

蒸気圧 > 0.01 mmHg

気液界面で気相濃度を稼げる揮発性。

水中の匂いがなぜ嗅げるか — ヘンリーの法則

— Henry's Law (1803) —
C気相 = KH · C水中 (1.1)
KH: ヘンリー定数 (物質ごとに固有)
C水中: 水中濃度 / C気相: 気相平衡濃度
→ KH が大きいほど "嗅ぎ取りやすい" 物質！

つまり、水中濃度が同じでも KH が大きい分子ほど匂う。逆に水溶性が極端に高い物質 (KH 小) は、水中にどれだけあっても気相に出てこない。塩のイオンは KH ≈ 0 なのでまったく匂わない。

嗅覚閾値 (OTC) — 桁外れの感度

ヒトが匂いを検知できる水中濃度を 嗅覚閾値 (OTC: Odor Threshold Concentration) と呼ぶ。これが驚くほど低い。

▶ 主要匂い物質の水中嗅覚閾値
物質分子式M [Da]OTC [μg/L]匂いの感じ
ジオスミンC₁₂H₂₂O1820.01湖の青臭さ
2-MIBC₁₁H₂₀O1680.015カビ臭
硫化水素H₂S340.5卵が腐った匂い

水道法の残留塩素 ≥ 0.1 mg/L = 100 μg/L は、HClO の OTC とほぼ一致している。
つまり「殺菌に必要な濃度」と「人間が匂いを感じる濃度」は、ほぼ同じレベルに設計されている！

CHAPTER 02

分子⇌イオン
酸塩基平衡で匂いは動く

EP.01 では「分子は匂う、イオンは匂わない」と教わった。化学Ⅱでは、その境界が pH で連続的に動く ことを学ぶ。これが「処理水の匂い問題」の核心。

なぜイオンは気化できないか — 水和エンタルピー

イオンが水中にいるとき、極性をもつ H₂O 分子がイオンの周りに整列して水和殻を作る。一個あたりの水和エンタルピー ΔHhyd はおおむね −300 〜 −500 kJ/mol。一方、中性分子 (例: NH₃) の蒸発エンタルピー ΔHvap はせいぜい 20〜40 kJ/mol。

物理イメージ
イオンを気化させるには、水和殻 (300 kJ/mol級) を破壊する必要がある。これは共有結合 1 本を切るのと同じレベル。室温では事実上不可能。
→ だからイオンは水の中から出られない！

本丸: 次亜塩素酸の酸塩基平衡

後段塩素として水に入れる次亜塩素酸 (HClO) の解離は本特集の核心式。これが pH によって匂いが動く 理由を説明する。

— HClO の解離平衡 (pKa = 7.53) —
HClO ⇌ H⁺ + ClO⁻ HClO = 揮発する分子 (匂う) / ClO⁻ = イオン (匂わない)
pH = pKa + log10([ClO⁻] / [HClO]) (2.1) Henderson-Hasselbalch
これを変形 → αHClO = 1 / (1 + 10pH−pKa)
pH 低 (酸性) → HClO 優勢 → よく匂う
pH 高 (アルカリ) → ClO⁻ 優勢 → ほぼ無臭

CHAPTER 03

RO のしくみを
本気で理解する

EP.01 で見た「あなが 0.1nm のすごいフィルター」は、本当は ザル (サイズ排除) じゃない。化学Ⅱ で学ぶ 浸透圧 と 溶解拡散モデルで動いている。

① 浸透圧 — van't Hoff の式

半透膜を挟んで純水と塩水を置くと、純水側から塩水側へ水が流れ込む。これを 浸透 (Osmosis) という。止めるのに必要な圧力が 浸透圧 Π。van't Hoff の式で記述される。

— van't Hoff (1886) —
Π = i · C · R · T (3.1)
Π: 浸透圧 [atm] / i: ファントホッフ係数 (NaCl ≈ 1.9)
C: モル濃度 [mol/L] / R = 0.082 L·atm/(mol·K) / T: 絶対温度 [K]

これを 逆向き に使うのが RO。Π を超える圧力で水を「絞り出す」。海水 (0.6 mol/L NaCl, 25°C) で Π ≈ 28 atm、実用 RO は 50〜80 atm で運転する。

② 膜の透過機構 — Solution-Diffusion モデル

RO 膜は単なるザルではなく、水と溶質が膜に "溶け込んで"、"拡散して"、反対側に出る という機構で分離している。

— Solution-Diffusion Model —
J水 = A · (ΔP − Δπ) (3.2a)
J溶質 = (D · K / L) · ΔC (3.2b)
水フラックスは圧力差で動く / 溶質フラックスは濃度差で動く
→ 加圧すればするほど水ばかり通り、溶質は通らない！
K (分配係数) = 膜への溶け込みやすさ
D (拡散係数) = 膜内の動きやすさ

イオンは K が極端に小さい (膜の電荷で反発される)。大分子は D が小さい (動けない)。一方 水分子に近いサイズの中性小分子 (CO₂, NH₃, H₂S, HClO, 小さな VOC) は、水と類似の挙動で膜を通り抜けてしまう。

③ 分子量 vs 阻止率 — 理論曲線

📊 分子量と阻止率の関係 (理論)

▶ 中性分子は S 字カーブで、~150 Da 付近を境に阻止率が急変
イオンは電荷反発で分子量に関係なく高阻止率を維持！
けっか
「ほぼ通さない」が成立するのは分子量 500 Da 以上、または イオン。
ジオスミン (182 Da) や 2-MIB (168 Da)、HClO (52 Da) のような 嗅覚的に最重要な小分子は通ってしまう。だから多くの家庭用 RO 浄水器は、後段に活性炭ポリッシャーを置く！
CHAPTER 04

純水のパラドックス
「純粋すぎて反応的」

RO 透過水はイオン総量 (TDS) が原水の 1〜5% に落ちる。一見「無垢で安全」だけど、化学的には 逆に不安定。これがあとの匂い問題に効いてくる。

① CO₂ 吸収で pH が急変する

純水は緩衝能 (buffer capacity) がほぼゼロ。空気と接触するとヘンリーの法則に従って CO₂ を吸う。

— 大気平衡 CO₂ 吸収 (pH ≈ 5.6) —
CO2(g) ⇌ CO2(aq)
CO2(aq) + H2O ⇌ H2CO3 ⇌ H⁺ + HCO3− → 平衡 pH ≈ 5.5〜6.0

前章の Henderson-Hasselbalch で計算すると、pH 5.6 では αHClO ≈ 99%。純水に塩素を入れた瞬間、ほぼ全量が揮発性側に振れて匂いがキツくなる。これが RO 水特有の「匂いやすさ」の構造的原因。

② 純水は容器を侵食する

純水の溶媒和力 (なんでも溶かしたがる傾向) は、容器・配管素材から微量成分を引き出してしまう。

PE/PP

残留添加剤・酸化分解物

いわゆる「ビニール臭」「プラ臭」。

PVC

フタル酸エステル類

独特の薬品様の匂い。

金属

Fe²⁺ / Cu²⁺ (低pH下)

「鉄っぽい味」(実は脂質酸化媒介)。

ふしぎ
半導体工場の 超純水 (UPW) ラインは、ステンレスやガラスすら溶かすので PVDF や PFA を使う。「純粋であること」と「化学的に安定であること」は別物！

③ TOC が低くてもバイオフィルムは形成される

TOC (Total Organic Carbon) が 1 mg/L 未満でも、Pseudomonas など貧栄養細菌は増殖できる。タンク内壁にバイオフィルムが形成されると嫌気状態が局所的に生まれ、以下の代謝産物が発生する。

— 嫌気代謝による匂い物質生成 —
SO42− + 有機物 → H2S + ... 硫酸塩還元菌 (SRB) — 卵腐敗臭
アミノ酸残基 → R-NH2 (アミン類) タンパク質分解 — 生臭・魚臭
CHAPTER 05

後段塩素の化学
DBPとクロラミン

業務用 RO 浄水システムの多くは、配送・保管中の細菌再増殖を防ぐため RO 後段で塩素を再添加 する。これは 設計上の選択。

— 次亜塩素酸ナトリウム投入時 —
NaClO + H2O → Na⁺ + ClO⁻
ClO⁻ + H⁺ ⇌ HClO 中性 pH では HClO/ClO⁻ が共存 (CH2 で見た平衡)

DBPs — 消毒副生成物の連鎖

HClO は強い酸化剤・求電子剤として、水中の有機物・アンモニアと反応する。これが DBPs (Disinfection By-Products: 消毒副生成物)。

1. ハロホルム反応 → トリハロメタン (THM)

フミン酸など腐植物質中のメチルケトン構造に HClO が逐次置換し、CHCl₃ などを生成する。

— ハロホルム反応 (Haloform Reaction) —
R-COCH3 + 3 HClO → R-COCCl3 + 3 H2O
R-COCCl3 + H2O → R-COOH + CHCl3 クロロホルム (THM) 生成! プールの底っぽい匂い

2. クロラミン階段的生成 (Cl/N 比依存)

原水中のアンモニアや有機アミンと HClO は段階的に反応する。pH と Cl/N モル比で生成物が変わる。

— Chloramine 段階生成 —
NH3 + HClO → NH2Cl + H2O モノクロラミン (プールのカルキ臭の主犯)
NH2Cl + HClO → NHCl2 + H2O ジクロラミン (魚生臭・刺激臭)
NHCl2 + HClO → NCl3 + H2O トリクロラミン (高揮発・目刺激)
ふしぎ
「プールの塩素臭」と俗に呼ばれる匂いの大半は、HClO 単独ではなくクロラミン類！
「塩素を入れすぎたから匂う」のではなく「不純物 (尿素・汗・皮脂) と塩素が反応した結果」というのが化学的真実。

Breakpoint Chlorination — 不連続点塩素処理

Cl/N モル比を 1.5〜2 まで上げると、クロラミン類は分解して N₂ として揮発し、遊離塩素 (HClO) のみが残る。これを Breakpoint Chlorination という。上水処理ではこの操作で「匂いの少ない殺菌力」を得るが、塩素投入量は多くなる。

CHAPTER 06

考察問題ラボ
化学で水を見破る

本特集の集大成。3 問の考察問題 で、化学Ⅰ・Ⅱ の式と概念を使って実際の水質問題を解いてみよう。解答は黒板にチョーク書き風で出る。

📝 PROBLEM 1 — van't Hoff の式

25°C, NaCl 0.05 mol/L の食塩水の浸透圧 Π を求めよ。さらに、これを RO で淡水化するのに必要な最小圧力を考察せよ。
(ファントホッフ係数 i = 1.9, R = 0.082 L·atm/(mol·K))

解答: Π = i · C · R · T を代入する。T = 298 K だから、 Π = 1.9 × 0.05 × 0.082 × 298 ≈ 2.32 atm RO で淡水化するには Π を超える圧力が必要。実際は流速確保のため Π の 2〜3 倍 (約 5〜7 atm) で運転する。
発展: 海水 (0.6 mol/L NaCl) なら Π ≈ 28 atm、実運転は 50〜80 atm。海水淡水化が高エネルギー消費な物理化学的根拠はここ。

📝 PROBLEM 2 — Henderson-Hasselbalch

RO 処理水の pH = 6.0、全塩素濃度 1.0 mg/L (Cl 換算) のとき、[HClO] と [ClO⁻] の濃度を求めよ。さらに、HClO の嗅覚閾値 OTC = 0.1 mg/L の何倍になるか答えよ。
(pKa = 7.53)

解答: αHClO = 1 / (1 + 10pH−pKa) より、 α = 1 / (1 + 10−1.53) ≈ 1 / 1.0295 ≈ 0.971 よって [HClO] = 1.0 × 0.971 ≈ 0.971 mg/L、[ClO⁻] = 1.0 × 0.029 ≈ 0.029 mg/L。
OTC 倍率 = 0.971 / 0.1 ≈ 9.7 倍。
解釈: やや酸性の RO 水に法定下限 1 mg/L の塩素を入れただけで、嗅覚閾値の約 10 倍。匂いをハッキリ感じるレベル。pH 8.5 まで上げれば α ≈ 0.097 で OTC とほぼ同等になり ほぼ無臭に。

📝 PROBLEM 3 — 純水のパラドックス

純水を空気中に放置すると CO₂ を吸って pH 5.6 まで下がる (大気平衡)。この pH における [HClO] : [ClO⁻] のモル比を求め、塩素を投入する際の匂いへの影響を化学的に説明せよ。(pKa = 7.53)

解答: log10([ClO⁻]/[HClO]) = pH − pKa = 5.6 − 7.53 = −1.93 [ClO⁻] / [HClO] = 10−1.93 ≈ 0.0117 よって [HClO] : [ClO⁻] ≈ 98.8 : 1.2。HClO がほぼ全量。
解釈: 純水は大気接触だけで自然に酸性側に振れる。後段塩素を入れた瞬間に揮発性の HClO がほぼ 100% になり匂いがキツくなる。これが 「RO 水なのに匂う」現象の構造的原因の 1 つ。
対策: ① 脱気済み密閉ライン / ② pH 緩衝剤を加える (重曹等) / ③ 後段活性炭でポリッシュ / ④ 使用直前に汲んで揮発させる。

切り分けの実証プロトコル

家庭・教室でできる検証順序。市販の DPD 試薬 (薬局・浄水器ショップで購入可) でかなり絞れる。

STEP 1: pH 測定

リトマス紙でも pH メーターでも可。
pH 6 台 → HClO 優勢で匂いやすい / pH 8 台 → ClO⁻ 優勢で匂いにくい。
pH 6 未満なら CO₂ 過剰吸収 or 配管酸性化を疑う。

STEP 2: DPD 試薬で遊離・結合塩素を測定

DPD 第 1 試薬 → 遊離塩素 (HClO + ClO⁻) が出る。
DPD 第 2 試薬 → 結合塩素 (≒ クロラミン類) が出る。
遊離が高い → HClO 由来 / 結合が高い → クロラミン由来 (NH₃ 残存を疑う)。

STEP 3: 1 時間放置揮発テスト

ガラス容器に汲んで空気にさらし、1 時間後に再度嗅ぐ。
匂いが弱まる → 揮発性 (塩素・VOC) / 残る → 非揮発成分 (溶出物・有機酸)。

STEP 4: ガラス vs プラ容器比較

同じ水を ガラス容器と プラスチック容器に汲み比べる。
プラ側だけ匂う → 容器素材溶出由来 (ビニール臭・薬品様)。

まとめ
① 匂い = 気相に出る中性分子、ヘンリー定数 KH で決まる
② pH によって 分子⇌イオン平衡 が動く (Henderson-Hasselbalch)
③ RO は 溶解拡散モデル、イオン・大分子を分けるが小中性分子は通す
④ 純水は 緩衝能ゼロ + 高溶媒和力で化学的に飢える
⑤ 後段塩素は DBPs (THM/クロラミン) を生む

「ほぼ通さない RO」と「処理水の匂い」のギャップは、化学を深く見るとパラドックスではなく必然になる！
