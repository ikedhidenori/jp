1. 4ブランド プレビュー

3. 使い方

3.1 最小 HTML テンプレ

<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>{{ title }}</title>
  <link rel="stylesheet" href="../_assets/design-system/design-system.css">
</head>
<body class="brand-innfra">
  <div class="container stack">
    <h1>{{ title }}</h1>
    <!-- ここにコンテンツ -->
  </div>
</body>
</html>

3.3 トークン圧縮戦略 (Thariq 論)

  • Claude Code は CSS を <link rel="stylesheet"> 外部参照させると、毎回 CSS を出力しない
  • 実測: 12,115 → 6,723 トークン (44%減)
  • 本ファイル (design-system.css) は 1度だけ 生成し、各 HTML artifact から相対パスで参照する
