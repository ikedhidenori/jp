Context

3. 業態 (What)

キュレーション型キャビン予約サイト + インフルエンサー・マーケティング事業 (ハイブリッド)

業態の core は下記 2 つの組み合わせ:

  1. B2C 予約キュレーション — Plum Guide 型 (上位 N% を厳選してリスト化、Cabin Fever Approved スタンプ付与)
  2. B2B コンテンツ・マーケティング受託 — 物件オーナー向けに Instagram 280K リーチを使った写真・動画・SNS 運用代行 (Inn Style パートナー情報で R8,000/marketing package と明示)

→ 単純な OTA / アフィリエイトではなく、メディア兼マーケットプレイス。

Cabin Fever は B 側 (物件オーナー) の「集客苦」を解決する代わりに、A 側自身は capex 軽い lifestyle business を成立させる 共生モデル。

日本 (屋久島スケール) への換算メモ

  • 土地: 南アより安い (山林) 〜 同等 (移住先人気エリア)
  • 建築: 日本の木造小規模 (40-60 sqm) は ¥30-50 万/sqm = ¥1,200-3,000 万 / 棟 (南ア比 1.5-2x)
  • ホットタブ・五右衛門風呂・薪ストーブ込みで Initial ¥1,500-3,500 万 / 棟
  • 稼働率は屋久島で 50-65% 想定可、客単価 ¥25-40K / 泊で Running 比率 50-60%
  • → 1 棟あたり ROI は南アと同水準 (1-15%)、絶対額は円ベースで小さい
  • スケーラビリティは Cabin Fever のように「複数棟キュレーション + メディア」で initial を分散する方が現実的

8. 差別化ポイント

軸 Cabin Fever の差別化
地理特化 南アフリカ Western Cape の hyper-local 専門。グローバル OTA (Airbnb / Booking) には模倣困難な土地勘
インフルエンサー資産 Instagram 280K+ という 獲得済オーディエンス = 新規参入の OTA より CAC が低い
キュレーション基準 創業者夫婦が 100+ 物件を実地調査 = "tried-and-tested" の信頼性スタンプ
デザイン感度 物件ラインナップが Scandinavian / 木造モダン系に統一 = 視覚的ブランド一貫性 (Instagram と相性◎)
B2B 受託の二毛作 キャビンオーナー = 個人副業層が多く、マーケ予算は持つが時間がない → ニッチ受託で stable cashflow
ライフスタイル事業性 Founders 夫婦 2 名 = 低固定費・高利益率。スケールよりも life quality 優先

  1. ライフスタイル事業の典型解 — 創業者夫婦 2 名で年商 R600K-1M 規模 (推計)、自社所有不要 = capex ゼロ、Instagram 資産が moat。屋久島 village 構想で参考になる「重資産を持たないキュレーション・メディア型」モデルの好例
  2. 資産形成は SNS 先行 — 2019 IG → 2022 事業化 = 3 年の信用蓄積後にマネタイズ という王道
  3. 二毛作モデルの強さ — B2C 予約だけだと OTA に勝てないが、B2B (オーナー向けマーケ受託) で安定収入を確保
  4. 地理的 moat — 南アという "微温市場" を独占。グローバル OTA はここまで土地勘を持たない
  5. スケール上限 — 創業者依存 (個人ブランド)。Plum Guide の 3% 仕組み化 / AfriCamps のフランチャイズ化を導入しない限り、年商 R5M 以上の壁は厚い

14. ジョニー提示の 5 特徴を言語化

ジョニー入力 (原文):
1. 大自然・絶景を「占有」
2. アウトドアリビング滞在の体験設計 (サウナ・プール・露天風呂・ソファ・焚き火・星空、大開口/窓全開で内外境界融合)
3. モダンラグジュアリー (唯一無二、冒険要素)
4. 小屋・トレーラーなど建築は最小
5. キャンプではない (サービスがちゃんとある)

これは業界用語で言うと 5 軸の交点 にいる:

ジョニー語 業界用語 (英) 業界用語 (和)
1. 絶景を占有 Landscape Exclusivity / Site-as-Amenity 「ロケーション独占型」
2. アウトドアリビング Outdoor-First / Indoor-Outdoor Continuum / Biophilic Living 「内外境界融合型滞在」
3. モダンラグジュアリー + 冒険 Adventure Luxury / Modern Wilderness Luxury 「冒険系モダンラグジュアリー」
4. 建築最小 Microarchitecture / Light-Footprint Architecture / Invisible Footprint 「ライト・フットプリント建築」
5. キャンプではない Hospitality-Grade Service / All-Inclusive / Auberge-style 「ホスピタリティ完備型」

コンセプトワード候補 (5 つの統合)

複数案を並べる。最後の "Landscape Sanctuary" が個人的推奨。

案 解説 適合度
Wild Luxe Sanctuary wild × luxury × sanctuary。冒険・贅沢・聖域 ★★★
Landscape Sanctuary 景観そのものが商品。建築は脇役 ★★★★★
Outdoor-First Resort アウトドアリビング起点 ★★★
Microarchitecture Retreat 建築最小の理念前面 ★★★
Adventure Aman (借用) Aman の冒険版 = 業界では既に通じる ★★★★
Untamed Boutique 野性 × 厳選 (Plum Guide 系語彙) ★★
Off-grid Auberge フランス語のホスピタリティ込み一棟 ★★★★

1 行コンセプト案

「建築は最小、景観は最大。サービスは完備、自然は野生。」

Architecture minimum, landscape maximum.
Service curated, nature wild.

これでジョニーの 5 特徴の全てが収まる。

15. 既存カテゴリとの境界線 (差別化)

既存カテゴリ 何が同じ 何が違う
グランピング (テント型) 屋外指向、自然立地 テント = 一時的、Storage 不可、サービスはセミセルフが多い。ジョニー案は 常設建築 + フルサービス
キャンプ場・オートキャンプ 自然立地 DIY 滞在、設備セルフ。ジョニー案は 滞在中にゲストが調理・設営しない
コテージ・貸別荘 (Airbnb/Vrbo 系) 一棟貸し、キッチン付き 立地は住宅地・近郊、設計は住宅。ジョニー案は 絶景独占 + 設計が滞在体験ファースト
ヴィラリゾート (Aman / Soneva) フルサービス、ラグジュアリー 建築が「マッシブ」、敷地内人口密度高い。ジョニー案は 建築が最小、人口密度極低
オーベルジュ (KAI / 星のや) 食 + 自然 + サービス 多くは旅館ベース建築 + 客室密度。ジョニー案は 独立棟 + 一棟貸し
ツリーハウス・ガラスポッド 建築最小、自然没入 キワモノ単発が多い。ジョニー案は 複数体験要素 (sauna/温泉/プール/焚き火) の統合設計
テントサウナ・銭湯系アウトドア アウトドア体験要素 宿泊なし。ジョニー案は 宿泊体験の一部

→ ジョニー案の独自性 = 「ヴィラリゾートのサービスを、ツリーハウスのフットプリントで、グランピングの開放感とともに、別荘の独占感で提供する」。境界線上で全要素を統合した、まだ普及していないニッチ。

21.4 経営者の意思決定インプリケーション

  1. 3 年以上の遅延 = ビジネスが成立しない可能性。Capex は今が(歴史的に)安く、2027 以降は構造的に高い
  2. 客単価設定はインフレ込みで初期から ¥50K+ を狙う (Camp Sarika $6,000 / Lindis NZ$2,000 を見るに国際的にはまだ余地ある)
  3. Solar + 蓄電池はインフレヘッジ。Capex 一括投下で 25 年の Opex を固定化
  4. 建築は最小 = Capex の制約条件 = 競争優位。マッシブ建築を作らないことが、原価インフレ時代の防御
  5. 「サブスク + 直予約 + ロングステイ」の 3 点で OTA 依存・季節依存を緩和

  7. Solar + 蓄電池は Capex 一括で長期 Opex を固定化するインフレヘッジ
