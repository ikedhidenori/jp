ChapterI

嗅覚化学の基礎What is "smell"? — A primer on olfactory chemistry

「匂い」という現象は、感覚としては素朴だが、化学的にはかなり 厳しい条件を満たした分子だけが嗅覚に届く。ヒトの嗅覚受容体 (OR) は GPCR ファミリーに属し、約 400 種が鼻腔上皮に発現している。受容体は気相中のリガンド分子と結合して初めて信号を出す——つまり、空気にまで漂ってこない物質は、どれだけ濃く水に溶けていても匂わない。

気相に漂える分子の条件は、おおむね次のとおり。

(i) 分子量 30〜300 Da。大きすぎると拡散しない。小さすぎると種が限られる。(ii) 電気的に中性。電荷を持つイオンは水和殻が厚く気化できない。(iii) 適度な疎水性 (logP > 1.5)。受容体の脂質ポケットに入る必要がある。(iv) 蒸気圧 P° > 0.01 mmHg。気液界面で平衡濃度を稼げる。

水中に溶けた匂い分子が、口腔・鼻腔に届く濃度はヘンリーの法則で記述される。

Cair = KH · Cwater (1.1)
KH: ヘンリー定数 [-] / Cwater: 水中濃度 / Cair: 気相平衡濃度

KH が大きい分子ほど、わずかな水中濃度でも気相に出てきて匂う。逆に水溶性が極端に高い (= KH 小) 物質は、水中に大量にあっても気相に出ない。塩化ナトリウムの解離イオン Na+·Cl− は KH ≈ 0 で、海水を嗅いでも食塩自体は匂わない理由はここにある。

ヒトの検出限界 (OTC: Odor Threshold Concentration) は驚くほど低い。次表は水中での閾値。

ChapterII

分子とイオンの境界The molecule-ion boundary — why ions cannot smell

同じ「塩素」でも、Cl2 分子と Cl− イオンでは挙動が全く異なる。前者は強烈なカルキ臭を放ち、後者は完全に無臭。この差を生むのは、水と分子の相互作用エネルギーである。

水和エンタルピーの壁

イオンが水中に存在するとき、極性をもつ H₂O 分子がイオンを取り囲み「水和殻」を形成する。Na+ 一個あたりの水和エンタルピー ΔHhyd ≈ −406 kJ/mol、Cl− なら −363 kJ/mol。一方、中性分子 NH₃ の蒸発エンタルピー ΔHvap はわずか 23 kJ/mol。イオンを気化させるには共有結合 1 本に匹敵するエネルギーが必要で、室温では事実上不可能だ。

結果、イオンは 水の中に閉じ込められたまま動かず、嗅覚に届かない。一方、中性で水素結合の数本程度しか持たない分子は、室温で十分な蒸気圧を確保できる。

酸塩基平衡が匂いを動かす

分子⇌イオンの平衡は、酸塩基反応で常に動いている。次亜塩素酸 (HClO) の解離平衡は本特集の核心だ。

HClO ⇌ H+ + ClO− pKa = 7.53 (25°C) — HClO は弱酸、解離は pH に強く依存

Henderson-Hasselbalch の式により、各種の分率は pH の関数として書ける。

pH = pKa + log10([ClO−] / [HClO]) (2.1)

変形すると HClO 分率 αHClO = 1 / (1 + 10pH−pKa)。pH が低い (酸性) ほど HClO 優勢 → 揮発し匂う。pH が高い (アルカリ) ほど ClO− 優勢 → イオン化して匂わない。

ChapterIII

逆浸透の物理化学Reverse osmosis — van't Hoff and the Solution-Diffusion model

逆浸透 (Reverse Osmosis, RO) を理解するには、まず 順浸透 (Osmosis) を思い出そう。半透膜を挟んで純水と塩水を置くと、純水側から塩水側へ水が流れ込む。これを止めるのに必要な圧力が浸透圧 Π で、van't Hoff の式で記述される。

Π = i · C · R · T (3.1)
i: ファントホッフ係数 / C: モル濃度 [mol/L] / R: 気体定数 8.314 J/(mol·K) / T: 絶対温度 [K]

海水 (NaCl 約 0.6 mol/L、25°C、i ≈ 1.9) で計算すると Π ≈ 28 atm。塩水を真水にする (= 逆浸透) には、この Π を超える圧力をかけて水を「逆向きに」絞り出す必要がある。実用 RO では 50〜80 atm の動作圧力が一般的。

膜の透過機構 — Solution-Diffusion model

RO 膜は単なる「ザル」(サイズ排除フィルター) ではない。水と溶質が 膜に溶け込み、拡散して反対側に出る という溶解拡散モデルで記述される。

J水 = A · (ΔP − Δπ)

J溶質 = (D · K / L) · ΔC (3.2)
A: 水透過係数 / D: 拡散係数 / K: 分配係数 / L: 膜厚 / ΔC: 濃度差

水フラックス J水 は加圧で線形に増えるが、溶質フラックス J溶質 は 濃度差に依存する (圧力ではない)。だから加圧するほど水ばかり通って溶質は通らない = 阻止率が上がる。これが RO の根本原理。

阻止率 R は次で定義される。

R = 1 − (C透過水 / C原水) (3.3)

R が物質によって大きく違う理由は、分配係数 K と拡散係数 D の差にある。K は膜への溶け込みやすさ、D は膜内での動きやすさ。イオンは K が極端に小さく (荷電膜と反発)、大分子は D が小さい (動けない)。一方、水分子サイズに近い中性小分子 (CO2, NH3, H2S, 小さな VOC) は 水と類似の挙動で膜を通り抜けてしまう。

SIDE NOTE — なぜ水だけが通るのか 水分子は強い水素結合ネットワークを介して「ジャンプ」する独特の輸送モードを持つ (proton-hop, Grotthuss 機構の類縁)。RO 膜のポリアミド層の中で水は連なって通れるが、他の分子は同じ経路を共有しにくい。これが水選択性の物理的起源。

ChapterIV

純水の「化学的飢え」The chemistry of hunger — why pure water is reactive

RO 透過水は、イオン総量 (TDS) が原水の 1〜5% にまで落ちる。一見「純粋」で安全に見えるが、これは 化学的にきわめて不安定な状態を意味する。純水は本来溶けていた成分を失い、その「飢え」を埋めるべく周囲から何かを引き出そうとする。

CO₂ 吸収による pH 急変

純水は緩衝能 (buffer capacity) がほぼゼロ。空気と接触すると、ヘンリーの法則に従って CO₂ を吸収する。

CO2(g) ⇌ CO2(aq)
CO2(aq) + H2O ⇌ H2CO3 ⇌ H+ + HCO3− 大気平衡で水中 [CO₂(aq)] ≈ 1.3 × 10−5 mol/L → pH ≈ 5.6

大気 (CO₂ 約 0.04%) と平衡した純水の pH は 5.5〜6.0 に下がる。CO₂ をさらに吸えば pH 5 を下回ることもある。前章の Henderson-Hasselbalch から、pH 5.6 における HClO 分率 α ≈ 99% となり、塩素を入れた瞬間にほぼすべて揮発性側に振れる。

容器素材の溶出

純水の高い溶媒和力は、容器・配管素材から微量成分を引き出す。代表例:

素材溶出物匂い影響
PE / PP残留添加剤、酸化分解物「ビニール臭」
軟質 PVCフタル酸エステル類独特の薬品様
金属配管 (低 pH 下)Fe²⁺, Cu²⁺ 等金属臭 (脂質酸化媒介)
シリコンパッキン低分子シロキサン無臭〜微

バイオフィルム形成

TOC (Total Organic Carbon) が 1 mg/L を切る純水でも、Pseudomonas 等の貧栄養細菌は増殖できる。タンク内壁にバイオフィルムが形成されると、嫌気状態が局所的に発生し、以下の代謝産物が発生する。

SO42− + 有機物 → H2S + ... 硫酸塩還元菌 (SRB) による嫌気代謝 — 卵腐敗臭
アミノ酸残基 → R-NH2 (アミン類) タンパク質分解 — 生臭・魚臭
SIDE NOTE — 純水が侵食するパラドックス 実験室で精製した超純水 (UPW) はステンレスやガラスすら微量溶かす。半導体製造ラインの UPW 配管は耐性に PVDF や PFA を使う理由がここにある。「純粋であること」と「化学的に安定であること」は別物。
ChapterV

後段塩素の化学Post-chlorination — DBPs, chloramines, and the smell of safety

多くの業務用 RO 浄水システムは、配送・保管中の細菌再増殖を防ぐため RO 後段で塩素を再添加する。これは設計上の選択であり、本来 RO 自体は塩素を除去している。後段塩素は次の反応で水中に展開する。

NaClO + H2O → Na+ + ClO− 次亜塩素酸ナトリウム → 次亜塩素酸イオン
ClO− + H+ ⇌ HClO 中性 pH では HClO/ClO⁻ が共存 (pKa = 7.53)

DBPs — 消毒副生成物の連鎖

HClO は強い酸化剤・求電子剤として、水中の有機物・アンモニアと反応する。これが Disinfection By-Products (DBPs) の正体。

1. トリハロメタン (THM) 生成

フミン酸など腐植物質中のメチルケトン構造に HClO が逐次置換し、最終的に CHCl3 等を生成。

R-COCH3 + 3 HClO → R-COCCl3 + 3 H2O
R-COCCl3 + H2O → R-COOH + CHCl3 ハロホルム反応 — クロロホルム生成

2. クロラミン階段的生成

原水中のアンモニア (or アミノ酸) と HClO の反応は、pH と Cl/N モル比に依存して段階的に進む。

NH3 + HClO → NH2Cl + H2O モノクロラミン (プールのカルキ臭の主因)
NH2Cl + HClO → NHCl2 + H2O ジクロラミン (魚生臭・刺激臭)
NHCl2 + HClO → NCl3 + H2O トリクロラミン (高揮発・目刺激)

「プールの塩素臭」と俗に呼ばれる匂いの大半は HClO 単独ではなくクロラミン類。これは「塩素入れすぎ」ではなく「不純物 (尿素・皮脂) と塩素の反応」の結果である。同じことが配送中 RO 水でも起き、わずかなアンモニア性窒素・有機アミンが残ると匂いが立つ。

SIDE NOTE — Breakpoint chlorination Cl/N モル比を 1.5〜2 に上げると、クロラミン類は分解し N2 として揮発、遊離塩素のみが残る。これを「不連続点塩素処理 (Breakpoint Chlorination)」という。上水処理ではこの操作で「ニオイのない殺菌力」を得るが、Cl 投入量が多くなる。

3. ハロ酢酸ほか

近年の研究で、ジクロロ酢酸・トリクロロ酢酸 (HAAs)、N-ニトロソジメチルアミン (NDMA) など、より毒性の高い DBPs が次々と同定されている。「塩素処理は完全に safe ではない」という認識は世界の水道行政の共通理解で、Activated Carbon やオゾン併用などで前駆体を減らす方向に動いている。

— PROBLEM 5.1 —

pH 6.0、全塩素 1.0 mg/L (Cl 換算) のとき、[HClO] と [ClO⁻] を求めよ。Henderson-Hasselbalch (pKa = 7.53) を用いること。さらに、嗅覚閾値 OTCHClO = 100 μg/L = 0.1 mg/L の何倍になるか答えよ。

αHClO = 1 / (1 + 10pH−pKa) = 1 / (1 + 10−1.53) = 1 / (1 + 0.0295) ≈ 0.971
[HClO] = 1.0 × 0.971 ≈ 0.971 mg/L
[ClO⁻] = 1.0 × 0.029 ≈ 0.029 mg/L
OTC 倍率 = 0.971 / 0.1 ≈ 9.7 倍。
解釈: pH 6.0 (やや酸性) の RO 水に法定下限 1 mg/L の塩素を入れると、嗅覚閾値の約 10 倍。匂いをはっきり感じる濃度。逆に pH 8.5 まで上げると αHClO ≈ 0.097 で OTC とほぼ同等になり、ほぼ無臭。

— PROBLEM 6.1 —

RO 透過率 60% のジオスミンが、原水で 30 ng/L 含まれている場合、処理水中のジオスミン濃度を計算し、嗅覚閾値 OTC = 10 ng/L と比較して匂いを感じるか判断せよ。

透過率 60% = 阻止率 R = 40% (誤読注意)。原問題は「透過率 = (1 − R)」とすると…通常用語で「透過率 60%」は 60% が膜を通る意味。
C透過水 = C原水 × 透過率 = 30 × 0.60 = 18 ng/L
OTC = 10 ng/L に対し 18 ng/L = 1.8 倍。明確に匂いを感じるレベル。
解釈: 藻類発生期の湖沼を原水とする場合、RO だけでは閾値超えのジオスミンが残り得る。後段活性炭が必須となる理由。RO の限界は、人間の嗅覚の感度の前では小さい。
— PROBLEM 6.2 —

純水を空気中に放置すると CO₂ を吸って pH 5.6 まで下がる (大気平衡)。この pH における HClO/ClO− 比を求め、塩素を投入する際の匂い影響を述べよ。pKa = 7.53 を用いよ。

log10([ClO−]/[HClO]) = pH − pKa = 5.6 − 7.53 = −1.93
[ClO−] / [HClO] = 10−1.93 ≈ 0.0117
つまり [HClO] : [ClO−] ≈ 98.8 : 1.2。HClO がほぼ全量。
解釈: 純水は大気と接触するだけで自然に酸性側に振れ、後段塩素を入れた瞬間に揮発性の HClO がほぼ 100% になる。「RO 水なのに匂う」現象の構造的原因の一つ。対策としては脱気済み密閉ラインまたはpH 緩衝剤を加える等の工夫が必要。

結論

「ほぼ通さない RO」と「実際にする匂い」のギャップは、化学を一段深く見ると パラドックスではなく必然として現れる。サイズだけのフィルターではなく溶解拡散モデルで動く膜は、小分子中性物質を通す。純水は化学的に飢えており、後段塩素は安全のため意図的に残される。

匂いは多くの場合、システムの故障ではなく 設計のトレードオフの現れだ。化学を知れば、何が trade-off になっているか、どこに改善余地があるかが見える。それが本特集の射程である。
