1. 年代で 4 人の別人がいる

Phase α (2008-2012) Twitter 黄金期 — 社会派思考即時テキスト

  • 長文 / 改行 / 体言止め少なめ
  • 「やっぱり」「どうかしてると思う」「は?」など議論口調
  • ハッシュタグ少、絵文字少
  • 引用例:

    「野菜の出来が悪かったら買わない、虫が食ってたらクレームつける。発注した量を用意できなければ取引停止する。放射能しかり、たべものを作る上でのリスクを農家が全て負うような今の仕組みはやっぱりちょっとどうかしてると思う。」 (2012-02、❤36 ♺147)

「あの（！）高木新平が、ついに、Facebookにまで、書かざるを得なくなっているほど、事態は、どうしても深刻みたいです。」 (2012-12、❤33 ♺145)

→ この時期の voice は今のジョニーとほぼ別人だが、「真剣に世界に怒っていた青年」の解像度が高い。「逃げていい」第 0 章 (前史) に丸ごと使える。

Phase β (2018-2020) バンライフ確立 — メディア露出キャッチー期

  • 短文 / ＼ ... ／ 見出し多用 / 絵文字解禁
  • 改行リズム + ハッシュタグ羅列
  • 「ようやく」「しこたま」「ふあああ」「ヤバイ」「捗る」
  • 引用例:

    「＼ #オフグリッド100%を目指す #キャンプ場 が #八ヶ岳 に爆誕❢ ／
    春先からしこたま準備していたものがようやくローンチ！東京から２時間圏内に、実験的なキャンプ場ができました。」 (2020-06、TW ❤55)

「呼ばれた気がしたので貼っておきます🚐住みたくなるメルセデス #バンライフ」 (2020-07、TW ❤53)

→ プロモーション + 物語化のバランス。媒体取材文体に近い。

Phase γ (2020-2022) 離婚 → 再起動 — 自虐転換期

  • 自分への声かけ、自分実況中継
  • 「おつかれちゃん俺」「駆け抜けたなー」「なんかスッキリしたなー」
  • 引用例:

    「黙ってても仕方ないので。渡鳥ジョニー、離婚しました。２度目...苦笑　バツニです！今回は、出張中に駆け落ち夜逃げ音信不通など、なかなかハードモードで...それもいつか笑い話になる日が来るのかなと。」 (2020-12-08、❤93)

「なんかスッキリしたなー。ありがとうついったとおまいら！」 (2020-12-08、4h 後、❤28)

→ これが「逃げていい」v2 の本命 voice。深刻 → 自虐 → 自分への労い、の 3 拍子。再現可能なジョニー固有のリズム。

  • 序数で連ツイ、長文も増える
  • 過去回想 + 現在実装の対比

口癖フレーズ

  • 「＼ ... ／」 (お知らせ見出しの基本形)
  • 「ようやく」 (達成報告のトリガー)
  • 「しこたま」 (副詞、強調)
  • 「やばない？」「ヤバイ」 (感嘆)
  • 「ふあああ」「ふあーー」 (感動)
  • 「捗る」 (作業進捗表現)
  • 「いやー」 (前置き、自虐の起点)
  • 「ほんとに」「マジで」 (強調)
  • 「みんな」 (呼びかけの起点)
  • 「〜しようぜ」 (呼びかけ)
  • 「おつかれちゃん俺」 (自分労い)
  • 「ありがとう (○○ + ついったとおまいら)」 (感謝定型)
  • 「謝罪行脚」 (2020-12 離婚発表の自己造語)

句読点・改行

  • 句点 (。) は段落末のみ、文中は読点 (、) 多用
  • 改行多め (2-3 文で 1 段落)、息抜きの空行
  • 絵文字は文末 に 1 個 (🚐 🌲 🧖‍♂️ 🍺 など装飾)

文末リズム (Twitter own tweets)

  • 体言止め (....で。/ ...と。) — 詠嘆・余韻
  • 「です/ます」 ← フォーマル文書
  • 「だ/だね」 ← 自己観察文
  • 「〜なる日が来るのかなと」「〜と思う」「〜じゃない？」 ← 推量・問いかけ
  • 「〜しようぜ」 ← 仲間への呼びかけ (「みんな #LivingAnywhere しようぜ」)
