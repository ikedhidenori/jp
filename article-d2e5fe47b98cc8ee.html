4. 素子推奨案 + 起動条件

推奨: Plan A 申請 + Plan B 同時準備 (A+B ハイブリッド)

理由:
1. Plan A 通過 = 戦略最適解。定期購読 ¥980/月の MRR 構造は MVP Gate ¥10 万達成の主軸。通過できるなら Plan A がベスト
2. Plan B は保険として低コスト。買切型マガジンの下書き作成は 1-2 時間の追加工数で済む。定期購読通過後は下書きを破棄するだけ
3. Plan C はブランド独立性毀損リスクが大きすぎる。戦略 v0.4.2 判断 G (新規開設) を覆すレベルの判断変更であり、Phase 2 での独立移行コストを考えると回避すべき。Plan B で 6/15 公開を担保できるなら Plan C を発動する理由がない

Plan C を発動すべき唯一のケース: Plan B でも公開できない致命的事態 (note アカウント自体が凍結される等、極めて低確率)
